== Description ==

This plugin adds a widget that displays a random flash card from a Pauker data set. See http://pauker.sourceforge.net/ for more details on Pauker.

== Installation ==

1. Upload `pauker.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Pauker widget in your sidebars and click '[EDIT]' to set the details.
